SEO Text Editor

Cross-checks keywords, indicates field length, SERP preview available

This SEO Text Editor enables editors to write consistent, SEO optimised texts.

The tool checks the field length of Title, Meta Description, Heading 1 and the Copy itself. Additionally, Keywords are cross-checked to make sure they appear consistently in the various fields. At the same time it ignores German and English Stop Words. Last but not least, a SERP preview is available.

Applied SEO rules

  • Common English and German stopwords are ignored.
  • Character and pixel count are validated and displayed above the input fields.
  • Keywords in the Title need to appear in the Text.
  • Keywords in the Title are not to be repeated in the Title.
  • Keywords in the Headline 1 need to appear in the Text.
  • 2-3 Keywords in the Meta Description need to appear in the Text.
